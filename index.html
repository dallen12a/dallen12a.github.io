3D Scene Render

C++ OpenGL

The project sets up an OpenGL environment, loads and renders a 3D scene with lighting, textures, and user interaction for navigating the scene and adjusting lighting properties. The main focus is on graphics rendering and interactive visualization using modern graphics techniques.

OpenGL Setup and Initialization:

The project uses OpenGL for rendering, and it starts by setting up OpenGL context using GLFW and GLEW libraries. It configures window dimensions, title, and sets up input callbacks.

Shader Programs:

The project defines vertex and fragment shader source codes, compiles them, and links them into shader programs. These shaders handle rendering operations, including lighting and texture mapping.

Mesh and Texture Loading:

The project defines structures and functions for loading meshes and textures. Textures are loaded from image files using the stb_image library.

Scene and Light Setup:

The project defines structures to hold information about the scene, meshes, lights, and cameras. Light properties (e.g., color, position) are initialized, including a spotlight and a key light.

Rendering and Input Handling:

The project implements functions for rendering the scene and processing user input. It uses camera controls for navigation (WASD, mouse) and allows changing the view mode (perspective/ortho). Lighting properties can be modified interactively via keyboard input (e.g., light position, color).

Main Loop:

The project enters a main loop where it processes input, updates the scene, and renders it.

Light Visualization:

The project includes visualization of the lights using simple geometric shapes (light cubes).

Texture Mapping:

Texture coordinates are used to map textures onto the meshes in the scene.
